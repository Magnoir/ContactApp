# Contact App

Contact App est une application Android développée en Kotlin qui permet aux utilisateurs de gérer leurs contacts.
Les utilisateurs peuvent ajouter, visualiser et supprimer des contacts.

## Installation

1. Clonez ce dépôt sur votre machine locale en utilisant `https://github.com/Magnoir/ContactApp` dans le cmd en se placant dans le bon dossier.
2. Ouvrez Android Studio et sélectionnez File puis Open et choisissez le dossier du projet cloné.
3. Attendez que Android Studio installe toutes les dépendances et synchronise le projet avec Gradle.
4. Une fois la synchronisation terminée, vous pouvez exécuter l'application en sélectionnant `Run > Run 'app'` dans la barre de menu en haut a droite.

## Utilisation

1. **Ajouter un contact** : Cliquez sur le bouton `Add Random Contact` pour ajouter un nouveau contact. Le contact est généré à partir d'une API et ajouté à la liste des contacts.
2. **Visualiser un contact** : Cliquez sur un contact dans la liste pour voir ses détails. Vous verrez l'image du contact, le nom, le prénom et le numéro de téléphone.
3. **Supprimer un contact** : Dans la vue de détail d'un contact, cliquez sur le bouton `Delete` pour supprimer le contact.
